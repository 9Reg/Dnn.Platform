Filtering Events by any Event Type

How to filter the event records that are displayed in the Log Viewer module by a single event type.

  1. Navigate to Admin >  Event Viewer - OR - Navigate to a Log Viewer module.
  2. Expand the Viewer section.
  3. At Type, select an event type from the drop down list. The matching records will then be displayed below.
