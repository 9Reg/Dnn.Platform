About Digital Asset Management Pro

The File Management page displays the Digital Asset Management Pro module that allows users to upload and manage site files. This Admin module is located on the Admin > Common Settings >  File Management page and can be added to additional pages. A separate version of this module that allows SuperUsers to manage host level files is located on the Host > Common Settings >  File Management page.

Installation Note: Digital Asset Management Pro is pre-installed on the site and can be added to site pages.
