About Device Preview Management

Device Preview Management allows Administrators and the Host to preview their mobile device site as it will appear on different mobile devices. A list of preview profiles for commonly used devices is provided for convenience and new preview profiles using either a pre-configured list of common profiles or you can create a new profile from scratch. Existing profiles can also be modified or deleted.

Device Preview Management is pre-installed on the Admin > Advanced Settings > Device Preview Management page and cannot be added to site pages.
