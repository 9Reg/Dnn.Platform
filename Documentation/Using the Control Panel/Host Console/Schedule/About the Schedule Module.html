About the Schedule

The Schedule page (Host > Advanced Settings Schedule) allows SuperUsers to schedule tasks to run against the DNN database at scheduled intervals. The Scheduler optimizes the efficiency of a site by reducing the frequency of common tasks. Note: The Schedule cannot be deployed to other site pages.

Important. The Schedule must be enabled and configured. See "Enabling/Disabling the Scheduler"
