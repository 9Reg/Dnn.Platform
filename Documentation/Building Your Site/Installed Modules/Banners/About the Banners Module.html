About Banners

Banners displays banner advertising that has been added via the Vendors page. Banners can be plain text, HTML, a static image, an animated image or they can execute a script such as JavaScript. You can set the number of banners to be displayed and modify the layout. If there are more banners available than is set to display in the module one time, then different banners are displayed each time the page is visited or refreshed.
