Setting the Visibility of Document Columns

How to choose which document columns are hidden or displayed. The Title, Owner, Category, Modified Date, Size (Kb) and Download Link columns are visible by default. The Clicks column only displays the number of clicks for documents which have link tracking enabled. I.e. The "Track Number Of Times This Link Is Clicked?" field is checked. Important. Documents cannot be downloaded if the 'Show Title Link' field is disabled and the 'Download Link' is hidden. In this scenario, the document titles and visible columns are displayed as static text to all users who can view the module.

  2. Select the Documents Module Settings tab.
  3. At Display Columns, at Visible to set a column as visible - OR - to set a column as not visible.
