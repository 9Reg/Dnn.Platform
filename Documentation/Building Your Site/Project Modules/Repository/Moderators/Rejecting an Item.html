Rejecting an Item

How to reject a Repository item awaiting moderation.

  1. Click the MODERATE button. The number of items awaiting moderation is displayed on this button.
  2. Optional. Click the VIEW FILE link to view the file.
  3. Click the REJECT link.
  4. In the Rejection Reason text box, enter the reason the item was rejected.
  5. Click the Send Rejection Notice button. This send a rejection email that includes your reason to the author.
  6. Repeat Step 2-5 to reject additional items.
  7. Click the Return button.
