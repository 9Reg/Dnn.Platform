Overview of the Nifty Toggle Repository

The Nifty Toggle Repository displays of list of files with a corresponding image. The Nifty Toggle is similar to the Default and Nifty skins however it includes a summary field. Each listing displays the item title, image, summary, file size, number of times the file has been downloaded, and the date and time when the item was last updated. Users can click the "Show Details" button to display the full description of the item. A link to download the file is displayed to users in the download role.

Categories, sorting ratings and comments are enabled on this skin.
