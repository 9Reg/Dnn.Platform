Enabling/Disabling the All Items Category

How to enable an All Items category to be displayed in the Category selection list of the Repository. All items within the Repository are automatically assigned to an All Items category. If this option is disabled, items that are not assigned to another category cannot be viewed. If this option is disabled it is recommended that at least one category is created.

  2. Select the Repository Settings tab.
  3. At Include All Files, select one of the following options:
    • to enable the All Items category.
    • to disable the All Items category.
