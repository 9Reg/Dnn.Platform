Editing any Forum User's Signature

How to edit any Forum user's signature.

  1. Search and located the required user. See "Searching Forum User Accounts"

  1. Locate the required user and click the Edit  button beside their username. This displays the Edit Profile page.

  1. In the Signature text box, add/edit the user's signature. Basic HTML formatting may be disabled. See "Editing Global User Settings"
  2. Optional. Click the Preview link to preview the signature. This displays a preview of the signature on this page.
    1. Click the Edit link to edit this signature.
