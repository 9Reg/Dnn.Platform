Editing your Forum Signature

How to edit your forum signature.

  1. Select My Settings from the module actions menu - OR - Click the My Settings link. This opens the User Settings page. The Control Panel > Overview window is displayed.
  2. In the right-hand panel, select Profile > Edit Signature. This displays the Edit Signature page.

  1. In the Signature text box, add/edit your signature. Basic HTML formatting may be disabled.
  2. Optional. Click the Preview link to preview the signature. This displays a preview of the signature on this page.
    1. Click the Edit link to edit this signature.
