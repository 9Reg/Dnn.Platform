Splitting a Thread

How to split a forum post from its current thread and make it into a new thread. Moderators require moderate permissions on the particular forum to split a thread. See "Setting Forum Permissions"

  1. Locate and open the required post. See "Viewing any Post"

  1. Click the Split link beside the post. This displays the Split Thread page.
  2. At Splitting Post, you can view the title of the first post of the new thread being created by splitting this existing one
  3. At New Forum, associated with the new forum for this split thread.
  4. Optional. At Email Post Authors, to notify all users participating in this thread that this thread has been moved - OR - for no email notification.
  5. Optional. At Include Post In New Thread, to include other posts in this thread - OR - to only include the post selected for splitting. E.g. The first post listed here.
  6. Click the Split Thread link. This displays the message "All Posts which are not approved and are being split into the new thread will be approved during the split!"
