Setting Thread Status

How to set the status of a forum thread. This is a useful housekeeping tool that helps users quickly identify when a post is informative or when a question is or isn't answered. Note: This option may be disabled.

  1. Locate and open the required post. See "Viewing any Post"
  2. At the (None Specified) drop down list, select either None Specified, Unresolved, Resolved, or Informative.
