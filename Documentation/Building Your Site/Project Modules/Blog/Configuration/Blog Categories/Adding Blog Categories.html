Adding Blog Categories

How to add blog categories.

  1. Select Edit Categories from the Blog_Categories module actions menu.
  2. In the Required. Enter the name of your category text box, enter the category name.

  1. Optional. Select a parent category from the drop down list if this new category should become a child category (sub-category) of an existing category.
  2. Click the Add link. The category is now listed in the Portal Categories list displayed on the right of this page.
