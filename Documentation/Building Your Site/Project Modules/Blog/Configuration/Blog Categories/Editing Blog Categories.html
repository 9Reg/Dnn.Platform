Editing Blog Categories

How to edit blog categories.

  1. Select Edit Categories from the Blog_Categories module actions menu.
  2. In the Portal Categories list (displayed on the right of this page) click on the category to be edited.

  1. In the Required. Enter the name of your category text box, edit the category name.
  2. At the drop down list, select a new parent for this category - OR - select None to make this a parent category.
