Modifying Recent Comments Template

How to modify the template which defines the content, layout, style and number of blog comments.

  1. Go to Recent_Comments.
  3. Select the Recent Blog Comments Settings tab.
  4. In the Display Template text box, modify the HTML template as required.
  5. In the Maximum Comments text box, modify the maximum number of comments to be displayed. The default setting is ten (10) comments.

Here is the default Display Template for comments:
