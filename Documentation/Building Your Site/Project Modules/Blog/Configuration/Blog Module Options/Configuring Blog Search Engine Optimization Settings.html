Configuring Blog Search Engine Optimization Settings

How to configure search engine optimization settings for blogs.

  1. Select Module Options from the View_Blog module actions menu.
  2. Go to the SEO Settings section.
  3. At Show Unique Title, to display the blog title in the Web browser - OR - to display the page title.
  4. At Show SEO Friendly URL, select from these options:
    • to show search engine friendly URL's. This modifies the URL to replace default.aspx with the title of the blog entry as follows: .../Blog-Entry-Title.aspx. This URL format can help with search engine optimization.
    • to disable.
  5. At Manage Entry Links, click the Regenerate all Permalinks for this Module link. You would typically perform this task after a change has been made to the URL format for a site.
