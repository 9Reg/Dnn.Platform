Configuring RSS Settings for Blogs

How to configure RSS (really simply syndication) settings for blogs. Note: Syndication must be enabled on individual blogs.

  1. Select Module Options from the View_Blog module actions menu.
  2. Go to the RSS Settings section.
  3. At Show Unique Title, to display the blog title in the Web browser - OR - to display the page title.
  4. At Include Body, select from these options:
    • to include the body of blog posts if it requested by the RSS feed.
    • to exclude the body of blog posts regardless of whether they are requested by the RSS feed.
  5. At Categories in Description, select from these options:
    • to include a text underneath the summary of the post with the categories.
    • to disable.
  6. At Tags in Description, select from these options:
    • to include a text underneath the summary of the post with the tags.
    • to disable.
  7. In the RSS Cache Time text box, enter the number of minutes until cached RSS feeds are refreshed. The default setting is 10.
