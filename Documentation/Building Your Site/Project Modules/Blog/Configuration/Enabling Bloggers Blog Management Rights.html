Enabling Bloggers Blog Management Rights

How to enable users to add blog entries, manage settings for their own blog, add mass edit their blog posts. Note: Bloggers must be viewing their own blog to access these tools. Bloggers can also access these tools using the New_Blog or View Blog.

  1. Go to the View_Blog module.
  3. Select the Permissions tab.
  4. At Edit Module (DNN Platform) or Full Control (Evoq Content and Evoq Content Enterprise), select the roles/usernames who are authorized to create a blog.
