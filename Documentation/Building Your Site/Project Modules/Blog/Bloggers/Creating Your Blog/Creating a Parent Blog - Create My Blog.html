Creating a Parent Blog - Create My Blog

How to create your first blog using the New_Blog. This blog will be a parent blog under which you can create multiple child blogs. Each of your blogs can be set as either public (can be viewed by others) or private (only you can view private blogs). Only authorized users can create blogs. See "Enabling Users to Create Blogs"

  1. Click the Create My Blog link on the New_Blog - OR - Select Create Blog from the Blog_List module actions menu. This opens the Create New Blog page.
  2. In the Blog Options section, complete these settings:
    1. In the Title text box, enter a title for the blog. This title is displayed in the Blog_List.
    2. In the Description text box, enter a description of the content of your blog.
    3. Optional. At Make This Blog Public, select from these options:
      • to make this blog viewable to all users who are authorized to view this module.
      • to set this blog as private. Private blogs can only be viewed by the blog owner. I.e. They cannot be viewed by any other user including Page Editors and Administrators.


  1. The blog name is now listed in the Blog_List and the blog is displayed in the View Blog. You can now add your first blog entry.
