Deleting all Unapproved Comments

How to delete all unapproved comments from blog entry. This task can be performed by bloggers on their own blog or by Administrators on all blogs.

  1. Locate the required blog entry. See "Viewing Blog Entries for a Month", See "Viewing Blog Entries for a Single Date", or See "Viewing Recent Entries for a Single Blog".
  2. Click the  Comments (1) link. This displays the comments associated with the selected entry.  All unapproved comments display the  Approve link.

  1. Click the  Delete all unapproved comments link located above all the comments associated with this entry.
