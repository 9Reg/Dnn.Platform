Approving a Blog Comment

How to approve blog comments. This task can be performed by bloggers on their own blog as well as by the Administrator on all blogs.

  1. Locate the required blog entry. See "Viewing Blog Entries for a Month", See "Viewing Blog Entries for a Single Date", or See "Viewing Recent Entries for a Single Blog". 

  1. Click the  Comments (1) link.


  1. Click the Approve  button or the Approve link. Note: You cannot undo approval for a comment, however you can edit or delete it.

Tip: Bloggers can enable a notification email to be sent when comments are posted which helps you know where to locate comments requiring approval. See "Blog Settings (Bloggers)" for details on enabling comments and comment options.
