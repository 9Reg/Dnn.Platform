Unpublishing a Blog Entry

How to remove a blog entry from being visible to others on the site without deleting it. This task can be performed by bloggers on their own blogs and by Administrators on all blogs.

  1. Locate the required blog entry. See "Viewing Blog Entries for a Month", See "Viewing Blog Entries for a Single Date", or See "Viewing Recent Entries for a Single Blog".
  2. Click the  Edit Entry link.

  1. Click the Save and Take Offline button. This displays the message "This will save your changes and take the post offline. Other users will not be able to see it any longer. Do you wish to continue?"
  2. This displays the message "This entry is still a draft: it will not be visible to readers until it is published."

  1. Click the Cancel button to return to the module.
