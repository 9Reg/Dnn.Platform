Deleting a Blog Entry

How to delete a blog entry. This task can be performed by bloggers on their own blogs and by Administrators on all blogs.

  1. Locate the required blog entry. See "Viewing Blog Entries for a Month", See "Viewing Blog Entries for a Single Date", or See "Viewing Recent Entries for a Single Blog".
  2. Click the  Edit Entry link.


  1. Click the Delete button located at the base of this page. This displays the message "Are You Sure You Wish To Delete This Item?"
