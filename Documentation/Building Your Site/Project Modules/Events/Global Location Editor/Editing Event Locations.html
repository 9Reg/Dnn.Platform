Editing Event Locations

How to edit event locations.

  1. Select Edit Locations from the module actions menu.
  2. In the Location Name list, click on linked [Location Name] to be edited. This displays the location details in the edit fields to the left.
  3. Edit the Location and/or Map URL fields as required.
  5. Repeat Steps 2-4 to add additional locations.
  6. Click the Return link to return to the module.
