Editing a Product (Product Manager)

How to edit a product as a Catalog Manager using the Store module.

  1. Go to the Store page and use the Store Menu to navigate to and select the category where the product is located.
  2. In the Store Catalog module, locate the required product.
  3. Click the Edit  button beside the title of the required product.
  4. Edit one or more product details. See "Adding a Product Category" for details of available fields.
