Managing Taxation on Orders

How to enable and set or disable taxation for Store orders. This tutorial is for the default taxation provider. Taxation is added to an order when the customer proceeds to the checkout.

  1. Go to the Store Admin page.
  2. If the Store Info page isn't displayed, click the Store Info link.
  3. Maximize  the   Tax Provider Settings section.
  4. At Enable Tax, select from these options:
    • to enable taxation.
      1. In the Tax Rate text box, enter the taxation rate as a percentage. E.g. 10

    • to disable taxation.

  5. Click the Update Tax Settings link.
