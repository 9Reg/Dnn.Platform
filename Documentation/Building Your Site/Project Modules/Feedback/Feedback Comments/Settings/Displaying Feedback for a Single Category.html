Displaying Feedback for a Single Category

How to display Feedback Comments for a single category. You can add multiple Feedback Comments to a single page and set them each display a different category or you may choose to set user and role permission so different instances are displayed to different users.

Tip: Categories are managed under Feedback Lists on the Feedback module.

  1. Go to a Feedback Comments.
  3. Go to the Comment View Settings section.
  4. At Category, select the category name.
  5. Click the Update link.
